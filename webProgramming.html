All about Web Programming

- The world of web development is as wide as the internet itself. Much of our social and vocational lives play out on the internet, which prompts new industries aimed at creating, managing, and debugging the websites and applications that we increasingly rely on.

- Web development is a broad description of the tasks and technologies that go into creating a website. It can be as simple as making a static text-based website or as elaborate as developing an interactive dynamic website.

- Web Programming can be broken into two different categories: frontend (client-side) and backend (server-side). Frontend code executes on the user's computer.
